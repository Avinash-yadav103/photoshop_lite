## Getting Started
To get started with PhotoshopLite, follow these steps:
1. **Installation**: Ensure you have Python and the required dependencies installed. Refer to the `requirements.txt` file for a list of dependencies.
2. **Running the Application**: Use the command `python run.py` to start the development server. Access the application via your web browser at `http://localhost:5000`.

## Image Editing Features
PhotoshopLite offers a variety of image editing functionalities:
- **Upload and Preview**: Users can upload images and preview them in real-time.
- **Basic Edits**: Crop, rotate, and flip images as needed.
- **Adjustments**: Modify brightness, contrast, and saturation.
- **Filters**: Apply various filters such as grayscale, sepia, blur, and edge detection.
- **Annotations**: Draw shapes, add text overlays, and insert stickers or watermarks.
- **Undo/Redo**: Easily revert changes with the undo/redo functionality.

## Video Editing Features
The platform also supports video editing with features including:
- **Upload and Preview**: Upload videos and preview them instantly.
- **Editing Tools**: Trim, merge clips, and replace audio tracks.
- **Filters and Effects**: Apply filters similar to those available for images.
- **Text Overlays**: Add captions and text to video frames.
- **Format Conversion**: Convert videos between formats like MP4, AVI, and GIF.

## Advanced Features
PhotoshopLite includes advanced functionalities for enhanced editing:
- **Layer-Based Editing**: Similar to professional tools like Photoshop and GIMP.
- **AI-Powered Features**: Future enhancements may include background removal and style transfer filters.
- **Face Detection**: Automatically detect and blur faces for privacy.

## System Requirements
- **Operating System**: Windows, macOS, or Linux
- **Python Version**: 3.6 or higher
- **RAM**: Minimum 4 GB (8 GB recommended)
- **Storage**: At least 500 MB of free space for installation

## Troubleshooting
If you encounter issues while using PhotoshopLite:
- Ensure all dependencies are installed correctly.
- Check the console for error messages and refer to the documentation for guidance.
- Restart the application if it becomes unresponsive.
