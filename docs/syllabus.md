# CSET344 Course Syllabus

## Course Title: Image and Video Processing (CSET344)

### Course Description
This course provides an in-depth understanding of image and video processing techniques. Students will learn about various algorithms and methods used in the field, including spatial and frequency domain processing, image enhancement, and machine learning applications in image analysis.

### Course Objectives
Upon completion of this course, students will be able to:
1. Apply spatial domain techniques for image processing.
2. Implement edge detection and morphological operations.
3. Understand and utilize color models and feature extraction methods.
4. Apply frequency domain techniques for image analysis.
5. Evaluate compression methods and machine learning techniques in image processing.

### Course Topics
1. **Spatial Domain Processing**
   - Sampling and Quantization
   - Histogram Processing
   - Gray-level Transforms
   - Spatial Filters
   - Edge Detection

2. **Advanced Image Analysis**
   - Canny Edge Detection
   - Harris Corner Detection
   - Color Models
   - Hough Transform
   - Morphological Operations

3. **Frequency Domain & Compression**
   - Fourier Transform
   - Frequency Filtering
   - Optical Flow
   - Image Compression

4. **Machine Learning & Video Processing**
   - Face Detection
   - PCA Analysis
   - SIFT & HOG Features
   - Video Processing

### Practical Components
The course includes laboratory experiments that allow students to apply theoretical concepts in practical scenarios. These experiments cover:
- Image Enhancement Techniques
- Image Zooming and Cropping
- Image Restoration and Noise Reduction
- Image Compression Techniques
- Image Segmentation Methods

### Assessment Methods
Students will be assessed through:
- Assignments and Lab Reports
- Midterm and Final Examinations
- Project Work demonstrating practical applications of course concepts

### Recommended Texts
- Digital Image Processing by Rafael C. Gonzalez and Richard E. Woods
- Computer Vision: Algorithms and Applications by Richard Szeliski
